# Chat
This application wrote in java allow you to chat with everyone!

# Execution 

> First Step
> *Build* **Server.java** and **Client.java**
'''
javac Server.java
'''

'''
javac Client.java
'''

> Second Step
> *Run* **Server.java** and **Client.java**
'''
java Server.java
'''

'''
java Client.java
'''
